## Installation
Github:

    git clone https://github.com/I2rys/YesE2EEFile

## Usage

    node index.js <file> <encrypt/decrypt> <key1> <key2>

+ file - The file to encrypt.
+ encrypt/decrypt - Either you want to encrypt/decrypt the file content.
+ key1 - First key.
+ key2 - Second key.
